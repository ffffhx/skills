1. 账号注册
1.1 推荐使用 Google 邮箱注册
- 步骤：
1. 打开 Google 邮箱注册页面，填写相关信息创建新邮箱
1.2 手机号验证解决方案
- 接码平台推荐：
- https://hero-sms.com/cn
- 操作步骤：
1. 选择一个接码平台并注册账号
2. 充值平台余额
3. 选择美国手机号码
4. 复制手机号码用于 Claude Code 注册
5. 接收验证码并完成验证

3. 代理搭建

3.1 购买美国 VPS 服务器
- 推荐服务商：
- https://lisahost.com/
- 配置建议：
- 本人购买的88元版本，晚上会有些慢，其他正常
3.2 搭建代理服务
- 操作步骤：
1. 打开 Claude Code
2. 输入prompt："使用https://github.com/XTLS/Xray-core，帮我在这台美国 VPS 上搭建一个代理服务器"
3. Claude code 会给你一个链接，然后会给你推荐几个客户端，你可以直接让它帮你下载并安装，本人用的是V2RayU
4. 打开V2RayU，复制链接，导入服务器配置，配置客户端连接信息
- 注意事项：
1. 代理的客户端本人使用的是V2RayU(尝试过V2RayXS，但是不知道为什么不行)
2. 可能需要给终端也配置代理，具体怎么配置可以直接咨询AI
3. 如果你的电脑是MAC，要注意打开的V2rayU会在你的屏幕的右上角，如果你右上角的应用太多可能会看不到V2rayU，我当时遇到这样的问题，明明进程存在，但就是找不到，结果发现是太多被遮挡住了
