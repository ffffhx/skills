## Summary

- 发布 GLM-5V-Turbo，定位为“Vision Coding Model”（视觉编程模型）。
- 强调原生多模态：理解图像、视频、设计稿、文档布局等输入并用于编码/推理。
- 宣称在多模态编码、工具使用、GUI Agents 等核心基准上表现领先，强调视觉能力与编程能力的平衡。
- 提到针对 Claude Code、OpenClaw 等 Agent 场景做了深度适配与协同。
- 给出体验入口与文档链接：chat.z.ai、docs.z.ai（VLM/GLM-5V-Turbo 指南），以及试用申请表单。
